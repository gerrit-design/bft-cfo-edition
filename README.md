## Overview

This is a dedicated implementation of the BFT platform optimized for CFO-level financial reporting, forecasting, and strategic business intelligence.

## Features

- **Executive Dashboard** - CFO-level financial summaries
- **Strategic Planning** - Long-term financial forecasting
- **Variance Analysis** - Budget vs. actual analysis
- **Cash Flow Management** - Advanced cash position tools
- **Financial Reporting** - Multi-format reporting (PDF, Excel, web)

## Tech Stack

- React/Vite - Frontend
- Node.js/Express - Backend
- PostgreSQL - Database
- Advanced charting and reporting libraries

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
git clone https://github.com/gerrit-design/bft-cfo-edition.git
cd bft-cfo-edition
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
